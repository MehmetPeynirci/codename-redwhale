# Cinnet2

Godot 4 için başlangıç FPS projesi.

## Özellikler
- First Person Controller
- Mouse look (kamera + pitch clamp)
- Yürüme / koşma / zıplama
- İvme ve yavaşlama hissi
- Harekete bağlı headbob
- Sol tık ile mouse capture, ESC ile bırak

## Kontroller
- `WASD`: Hareket
- `Shift`: Koşu
- `Space`: Zıpla
- `Mouse`: Bakış
- `ESC`: Fareyi serbest bırak
- `Sol Tık`: Fareyi tekrar yakala

## Çalıştırma
Godot 4 ile proje klasörünü açıp `Main.tscn` sahnesini çalıştırın.
